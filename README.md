# 🚗 Car Price Prediction API

A production-ready REST API that predicts used car prices using a Random Forest regression model. Built with FastAPI, secured with JWT authentication, cached with Redis, and monitored with Prometheus + Grafana — fully containerized with Docker Compose and deployed on Render.

**[🔗 Live Demo](https://fastapi-project-lux4.onrender.com/docs)** | **[📦 GitHub](https://github.com/Ankur377823/fastapi-Project)**

---

## 📌 Features

- 🤖 **ML Inference** — Random Forest model trained on real-world car dataset with full preprocessing pipeline
- 🔐 **JWT Authentication** — Token-based auth on all prediction endpoints
- 🔑 **API Key Validation** — Secondary layer of access control via request headers
- ⚡ **Redis Caching** — Repeated predictions served from cache, skipping model inference
- 📊 **Prometheus + Grafana** — Real-time API monitoring and metrics dashboards
- 🐳 **Docker Compose** — One-command spin-up of the full stack
- ☁️ **Render Deployment** — Live deployment with managed Redis

## 🚀 Getting Started

### Prerequisites
- Docker + Docker Compose
- Python 3.11+ (for local dev)

### 1. Clone the repo

```bash
git clone https://github.com/Ankur377823/fastapi-Project.git
cd car-price-api
```

### 2. Set up environment variables

```bash
cp .env.example .env
```

Edit `.env`:

```env
API_KEY=your-api-key
JWT_SECRET_KEY=your-secret-key
REDIS_URL=redis://redis:6379
```

### 3. Train the model

```bash
pip install -r requirements.txt
python -m training.train_model
```

### 4. Start the full stack

```bash
docker compose up --build
```

| Service | URL |
|---|---|
| API Docs (Swagger) | http://localhost:8000/docs |
| Prometheus | http://localhost:9090 |
| Grafana | http://localhost:3000 |

## 🔑 Authentication Flow

### Step 1 — Get JWT token

```bash
curl -X POST http://localhost:8000/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "admin"}'
```

**Response:**
```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIs..."
}
```

### Step 2 — Call predict endpoint

```bash
curl -X POST http://localhost:8000/predict \
  -H "token: <your_jwt_token>" \
  -H "api-key: demo-key" \
  -H "Content-Type: application/json" \
  -d '{
    "company": "Maruti",
    "year": 2014,
    "owner": "First",
    "fuel": "Diesel",
    "seller_type": "Individual",
    "transmission": "Manual",
    "km_driven": 145500,
    "mileage_mpg": 55,
    "engine_cc": 1248,
    "max_power_bhp": 74,
    "torque_nm": 190,
    "seats": 5
  }'
```

**Response:**
```json
{
  "predicted_price": 485000.0
}
```

## 📡 API Reference

### `POST /login`
| Field | Type | Description |
|---|---|---|
| username | string | Admin username |
| password | string | Admin password |

**Returns:** `{ "access_token": "..." }`

---

### `POST /predict`

**Headers:**
| Header | Description |
|---|---|
| `token` | JWT access token from `/login` |
| `api-key` | Static API key from `.env` |

**Body:**
| Field | Type | Description |
|---|---|---|
| company | string | Car manufacturer (e.g. Maruti) |
| year | int | Manufacturing year |
| owner | string | First / Second / Third |
| fuel | string | Petrol / Diesel / CNG |
| seller_type | string | Individual / Dealer |
| transmission | string | Manual / Automatic |
| km_driven | float | Total kilometers driven |
| mileage_mpg | float | Mileage in MPG |
| engine_cc | float | Engine displacement in CC |
| max_power_bhp | float | Max power in BHP |
| torque_nm | float | Torque in NM |
| seats | float | Number of seats |

**Returns:** `{ "predicted_price": <float> }`

## 📊 Monitoring

- **Prometheus** scrapes metrics from `/metrics` every 15 seconds
- **Grafana** available at `http://localhost:3000` (default login: `admin/admin`)
- Add Prometheus as a data source in Grafana → create dashboards for request rate, latency, and error rate
